# Python To-Do List Application

Simple command-line to-do list app implemented in Python.

**Features**
- View tasks: Lists all current tasks with an index number so you can review what's pending.
- Add tasks: Prompts for a task and saves it after trimming leading/trailing whitespace.
- Remove tasks: Remove a task by entering its list number; the app validates your selection before deleting.
- Exit: Choose this to quit the application cleanly and return to your shell.


**Usage**
- Select an option from the menu to view, add, remove tasks, or exit the program.


**Files**
- `todo-list.py`: main program file
